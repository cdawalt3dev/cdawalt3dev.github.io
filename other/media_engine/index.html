Media Engine

MediaEngine, a custom Android app, lets the user assign tags to media files (movies, images, or audio files) to bring up later in a search. Files with matching criteria are displayed and can be played using the program's custom video/audio player or image viewer for even faster access (no need to waste time loading the system's default player/viewer). This app makes both file navigation and playback easy and fast on short notice.

The file extensions for including files and path(s) to search for files are customizable.

This app is for Android phones and tablets. It can resize its UI to fit a wide range of sizes and aspect ratios (landscape or portrait). It would need cleanup and stability improvements to be feasible for the app store.
