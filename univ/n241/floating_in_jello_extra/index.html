Hobbies

Programming

GameMaker

I started programming when I was introduced to GameMaker 7, a 2-dimensional game design program with beginners in mind. It supported drag and drop commands to convey basic actions, such as moving an in-game object or waiting for an event. Most actions could also be represented in a simplified version of Java. I learned how to script by translating actions into script.

Visual Basic

In my sophomore year in high school, I took a class in Visual Basic. The class used Visual Studio to drag and drop GUI elements to windows to create simple applications. The Visual Basic Language could make the application interactive, such as printing output to text fields or opening a new window.
