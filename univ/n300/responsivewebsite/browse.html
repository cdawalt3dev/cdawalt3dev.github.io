Visual Studio


Visual Studio is an industry-strength IDE primarily for Visual Basic, C#, and C++. It supports a wide variety of other languages as well, including Python, Java, Ruby, and PHP. The ASP.NET framework is supported for building websites. The Community version is available for non-commercial use.

Eclipse


Eclipse is an open-source IDE with versions for C++, Java, and several other languages. The Web Development version supports HTML, XML, Javascript, and CSS. Several versions come with Git integration. Eclipse is portable, capable of running off a flash drive without prior installation on the local machine.
